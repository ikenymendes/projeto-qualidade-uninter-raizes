# Projeto Multidisciplinar - Qualidade de Software 🌵
## Estudo de Caso: Rede Raízes do Nordeste

Este repositório contém os artefatos de teste e garantia de qualidade para o sistema de gestão da franquia Raízes do Nordeste.

### 🛠️ Tecnologias Utilizadas
- **Automação:** Selenium WebDriver + Python
- **Performance:** Apache JMeter (Planeamento de Carga)
- **Segurança:** OWASP ZAP (Análise de Vulnerabilidades)

### 📂 Estrutura do Repositório
- `/automation`: Scripts de teste funcional para o fluxo de checkout.
- `/performance`: Documentação da estratégia de testes de carga e escalabilidade.
- `/security`: Relatório de conformidade de segurança e privacidade (LGPD).

### 🚀 Execução
Para validar o ambiente de automação:
1. Instale as dependências: `pip install selenium webdriver-manager`
2. Execute o script: `python automation/test_checkout.py`
