# Projeto Multidisciplinar - Qualidade de Software 🌵
## Estudo de Caso: Rede Raízes do Nordeste

Este projeto apresenta a estratégia de QA e automação para o sistema de gestão da franquia Raízes do Nordeste, garantindo a integridade dos dados e a performance do checkout.

### 🛠️ Tecnologias Utilizadas
- **Automação:** Selenium WebDriver + Python
- **Performance:** Apache JMeter (Planeamento)
- **Segurança:** OWASP ZAP (Auditoria)

### 📂 Estrutura do Repositório
- `/automation`: Scripts de teste funcional (Selenium).
- `/performance`: Estratégia de testes de carga.
- `/security`: Relatórios de segurança e conformidade.
